# GradeCalc

This is a simple C++ program designed to help learn the language

## Description

The program allows the user to enter module grades and credit values (with input validation) and then computes the final weighted grade once the user is finished entering grades.

## How to Build

Compile with clang++:

```
clang++ -std=c++17 grade.cpp -o grade 
```

## How to Run

After building, run the executable:

```
./grade
```

Enter numbers separated by spaces, then type '-1' to compute the grade.
